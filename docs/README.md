# Pixel Chart Diary

Daily asset status posts for the **Pixel Chart Diary** Facebook page.

We track US stocks and BTC using EMA-based signals, published daily with a simple emoji system anyone can read at a glance.

## What we post

| Post              | Content                                                                |
| ----------------- | ---------------------------------------------------------------------- |
| BTC Chart         | Pixel art chart — EMA 200/12/26 on 60-day candles + Claude AI analysis |
| Stock Status      | Multi-stock EMA regime table with emoji signals                        |
| Full Asset Status | Single-asset deep dive — Technical + AI Read + Onchain                 |

## Quick legend

```
📈🔵  AAPL   $200.00   +1.2%
```

| Part           | Meaning                          |
| -------------- | -------------------------------- |
| 📈 / 📉        | Bull or Bear regime              |
| 🔵 ⚪ 🔴 🟡 🟢 | Price position state             |
| ↑ ↓            | Cross direction (cross day only) |
